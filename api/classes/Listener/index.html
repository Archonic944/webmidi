Listener

Listener

The Listener class represents a single event listener object. Such objects keep all relevant contextual information such as the event being listened to, the object the listener was attached to, the callback function and so on.

  • Listener

    • new Listener(event, target, callback, [options])

    • .target : EventEmitter

    • .callback : function

    • .remove()

new Listener(event, target, callback, [options])#

Throws:

  • TypeError The `event` parameter must be a string or `EventEmitter.ANY_EVENT`.
  • ReferenceError The `target` parameter is mandatory.
  • TypeError The `callback` must be a function.
ParamTypeDefaultDescription
eventstring | EventEmitter.ANY_EVENTThe event being listened to
targetEventEmitterThe EventEmitter object that the listener is attached to
callbackEventEmitter~callbackThe function to call when the listener is triggered
[options]Object{}
[options.context]ObjecttargetThe context to invoke the listener in (a.k.a. the value of this inside the callback function).
[options.remaining]numberInfinityThe remaining number of times after which the callback should automatically be removed.
[options.arguments]arrayAn array of arguments that will be passed separately to the callback function upon execution. The array is stored in the arguments property and can be retrieved or modified as desired.

listener.event : string#

The event name

listener.target : EventEmitter#

The object that the event is attached to (or that emitted the event)

listener.callback : function#

The callback function

listener.context : Object#

The context to execute the context function in (a.k.a. the value of this inside the callback function)

listener.remaining : number#

The remaining number of times after which the callback should automatically be removed.

listener.count : number#

The number of times the listener function was executed

listener.arguments : array#

Arguments to pass separately to the callback function upon execution

listener.suspended : boolean#

Whether this listener is currently suspended

listener.remove()#

Removes the listener from its target.
